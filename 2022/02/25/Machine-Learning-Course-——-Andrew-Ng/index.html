Week One

  • Introduction

    • Supervised Learning：

      • Regression
        continuous valued output
      • Classifcation
        discrete（离散的） valued output

    • Unsupervised Learning

      • Clustering
      • Non-clustering
        聚类问题：算法不转换或改变数据（就像在基因问题中基因本身不会改变），它只是根据一些特征将它们分组（将基因问题分为不同种类或类型的基因） .
        然而，在非聚类问题中，算法正在转换数据/输入并改变数据。并且相信 Andrew NG 教授（作为他所在领域的顶级讲师和教授）的话，即鸡尾酒会问题是一个非聚类问题，我们现在可以明白为什么了，因为它改变了数据，在这种情况下，它改变了声音听起来很清楚，实际上它改变了两次以获得第一个声音，第二次获得第二个声音。
        在我看来，聚类问题最终的output是明确的几个类别，而非聚类问题则不是这样，它的定义很广泛，不属于聚类且无监督的不都是non-clustering吗

  • Model and Cost Function

    • 单变量回归问题y = θ_1x+θ_0y=θ1x+θ0y = θ_1x+θ_0y=θ1x**+θ0**平方误差函数J
      Squared error function or Mean squared error
      对于一般的情况，二变量的损失函数J的图像是这样的，那么初始值的选取会影响最终到达的驻点

    • 梯度下降α

      如果α小，它的移动很缓慢；但α大了，它却有可能错过最优解

      通过分析式子可以知道，α就是偏导数乘以的一个常量，即学习率，这样的方式可以让θ的取值不断接近最优解，直至J（θ）的偏导数为0，即到达了驻点（极值点或鞍点）或局部最优解，无论这个θ它的初始值是在左侧还是右侧，都是可行的。（这是对于单变量而言的）

  • Gradient Descent For Linear Regression

    对于线性回归函数而言，它的图像总是这样一个凹函数或弓形函数，所以用梯度下降法就可以很简单取到最优
    也叫“Batch” Gradient Descent 批处理梯度下降，因为每次下降，都得根据一批次的sample去检测loss

  • Feature Scalling

Where $ μ_i $ is the average of all the values for feature (i) and sis_isi​ is the range of values (max - min), or sis_isi​ is the standard deviation.

  • Learning Rate

    If α is too small: slow convergence.

    If α is too large: may not decrease on every iteration and thus may not converge(收敛).
