ios::showbase 为整数添加一个表示其进制的前缀
ios::internal 在符号位和数值的中间插入需要数量的填充字符以使串两端对齐

ios::showpoint 在浮点数表示的小数中强制插入小数点（默认情况是浮点数表示的整数不显示小数点）
ios::showpos 强制在正数前添加+号
ios::skipws 忽略前导的空格（主要用于输入流，如cin）
ios::unitbuf 在插入（每次输出）操作后清空缓存
ios::uppercase 强制大写字母

STL和基本的数据结构

  • 通用操作

    操作 描述
    X u; 创建名为u，类型为X的空对象
    X(a); 创建对象x的拷贝
    X u(a); 或 X u = a; u是a的拷贝（复制构造函数）
    r = a; r等于a(复制赋值)
    X u(rv);或 X u = rv; u等于rv的原始值（移动构造函数）
    (&a) -> ~X(); 对a的每个元素执行析构函数
    begin() 返回指向首元素的迭代器
    end() 返回指向超尾元素的迭代器
    cbegin() 返回指向首元素的const迭代器
    cend() 返回指向超尾元素的const迭代器
    size() 返回元素数目
    maxsize() 返回容器最大可能长度
    empty() 判断容器是否为空
    swap() 交换两个容器的内容
    == 若两个容器长度相同、元素相同、元素顺序相同则为真
    != 上述的反

    双向或随机迭代器的容器（vector, map, list, set, array, deque）是可反转的，则会提供可反转容器的操作
    （类似于尾指针的操作）

    操作 描述
    X::reverse_iterator 指向类型T的反向迭代器
    X::const_reverse_iterator 指向类型T的const反向迭代器
    a.rbegin() 返回指向a的超尾的反向迭代器
    a.rend() 返回指向a的首元素的反向迭代器
    a.crbegin() 返回指向a的超尾的const反向迭代器
    a.crend() 返回指向a的首元素的const反向迭代器

    更具体地内容请见C++ Primer Plus 附录G，需要默写
